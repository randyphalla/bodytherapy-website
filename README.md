# Body Therapy Website
> This final project is a thesis project for school for Interactive Media Specialist at Fanshawe College. The goal is to create a mobile or web application.

# Getting Started

##  Installing

`yarn` or `npm i` Once you done installing the `node_modules` run `npm start`

# Building

Run this command in the terminal `npm run build`. This command will build the project and be ready for deployment
